Flights.jar

A Flight program using google's QPX Express API
Flight program that looks up flight times and prices with a classic 'n-curses' overlay for Java. The program starts out with a simple flight search, runs the program and asks QPX for the Data. QPX then displays the top ten data entries in sorted by price and repeats for the return flight.

Future Expansion would include method for the user to pay for the flight data within the app or a link that would direct the user to the website where the user could purchase the flight that they were looking for.

Languages Used: Java
API's used: Google QPX Express
Libraries used: Lanterna

3D Pong

A 3d Pong game

A web application that utilizes three.js to create a 3d pong game. This game was created for my computer graphics class and had certain parameters. It had to contain music, bouncing sound effect, a 3d layout of the board, two paddles controlled with one controlled by a user and the other controlled by the CPU.

The ball movement is effected by a random number generator with the paddles adding to an x or y "spin".

Languages Used: Javascript, HTML, CSS
Libraries Used: three.js
